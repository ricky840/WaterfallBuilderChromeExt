Name Required.
Line Item Key Leave blank. Line item key does not exist yet. Required. This value should match to the line item key that need updates.
Order Key Required. Enter order key where line item should belong to. Leave blank. Otherwise value will be ignored.
Assigned Ad Unity Keys
Required. Comma separated. At least one ad unit key needs to be assigned.
Example) 860974aba50e432a8bd0eb9ba3717913,d33f6e6d816e45d49db410db22a55bee

Make sure you are assigning to an ad unit format that is supported by network. Otherwise, it will get an error. Not all formats are supported by networks. If you would like to see which format is supported by each network, see here.
Priority Required. Enter between 1 ~ 16 integer. Ex) 5
CPM Required. Enter between 0 < CPM < 5000. It can be entered upto two decimal point. Ex) 10.50
